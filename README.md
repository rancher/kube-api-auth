# kube-api-auth

A microservice for user authentication in kubernetes clusters.

## Building

`make`

## Running

`./bin/kube-api-auth`

For bugs, questions, comments, corrections, suggestions, etc., open an issue in rancher/rancher with a title starting with `[kube-api-auth]`.
